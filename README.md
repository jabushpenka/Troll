# Troll

## Что это?
Тролль — коллаборативная доска задач (почти Trello).

## Что это делает?
- Предоставляет пользователю систему досок (бордов), на которых можно располагать колонки.
- В колонки можно добавлять задачи в виде карточек.
- Карточки можно перетаскивать между колонками при помощи мыши.
- В карточки можно добавлять задачи и помечать их как выполненные (галочкой).
- Помимо этого, карточки имеют обширные опции форматирования (добавление меток, изменение цветов).
- На доску можно приглашать других пользователей, которые смогут на ней работать совместно друг с другом и автором.
- Обновления приходят в реальном времени, каждый пользователь может наблюдать за изменениями на доске без надобности в перезагрузке страницы.

## Зачем это?
Для отслеживания рабочих или бытовых задач, личных и совместных.

## Кому это нужно?
Наши пользователи — студенты, менеджеры, офисные работники.
Они хотят решить проблему отслеживания своих задач.
Их окружают коллеги или однокурсники, которые хотят иметь возможность отслеживать совместные задачи.
